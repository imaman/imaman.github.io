    Instrument

    Parameters

    • packageName: string | string[]

      the package name of the instrument. The package names allows logical grouping of related instruments: can be thought of as the "last name" of the instrument whereas the name (see _name) can be thought of as "first name". Package names are hierarchical: ["p1", "p2", "p3"] is nested inside ["p1", "p2"] which, in turn, is nested inside ["p1"]. If a string it is treated as a single element array, that is: "p1" is equivalent to ["p1"].

    • _name: string

      the "first name" of the instrument
